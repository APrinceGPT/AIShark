# AIShark Keyboard Shortcuts

Quick reference guide for keyboard shortcuts in AIShark.

## General Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+F` (Win) / `Cmd+F` (Mac) | **Focus Search** | Jump to packet search input field |
| `Esc` | **Close** | Close any open modal or dialog |
| `Ctrl+/` (Win) / `Cmd+/` (Mac) | **Show Help** | Display this keyboard shortcuts guide |

## Session Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+S` (Win) / `Cmd+S` (Mac) | **Save Session** | Save current analysis to database (requires login) |

## AI Features

| Shortcut | Action | Description |
|----------|--------|-------------|
| `A` | **AI Assistant** | Open AI chat interface for queries |

## Packet Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `N` | **Next Error** | Jump to the next error or retransmission packet |
| `P` | **Previous Error** | Jump to the previous error or retransmission packet |

## Tips

- **Search Focus**: Press `Ctrl+F` / `Cmd+F` at any time to quickly start searching packets
- **Quick Save**: Use `Ctrl+S` / `Cmd+S` to save your work without reaching for the mouse
- **Escape Everything**: Press `Esc` to close any modal - works on packet details, save dialog, shortcuts help, etc.
- **Error Hunting**: Use `N` and `P` to quickly navigate through problematic packets
- **AI on Demand**: Press `A` to instantly open the AI chat interface when you need analysis

## Accessibility

All interactive elements support keyboard navigation:
- Use `Tab` to move between elements
- Use `Enter` or `Space` to activate buttons
- Use arrow keys to navigate lists
- Screen reader support with ARIA labels
